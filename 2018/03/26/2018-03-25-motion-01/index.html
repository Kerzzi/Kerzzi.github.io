第0章 前言

过年期间不想浪费时间，于是我收集了我参投的项目以及市值排名前100名的项目的白皮书，准备一一分析。
本篇文章本想做一个Motion的项目分析，但是除了官网（http://motion.one/）和老版白皮书，没有发现太多有用资料，真是”云雾中的项目“，于是本文只对Motion这个项目做一个简单介绍。

第1章 MTN 如何分配

总量:21亿
IxO公开发行：4.2亿
私募：10.5亿
团队持有：6.3亿

第2章 Our Mission

Make your move, we will let you surprise!
这是官网上的显示的MTN的Mission，没找到太好的翻译（“行动吧，我们会让你吃惊？”），乍一看不知所云。

好在下面两句口号：

People who provide contribution can feel the value of Motion!
People who create value can enjoy the great return from Motion!

我们希望让那些对平台有所贡献的人能够感受到它的价值并享受Motion的巨大回报。

第3章 Motion基本信息

Motion is the first global blockchain-based game distribution platform of mainly video content and users’ recognition. Motion platform is dedicated to providing developers a game distribution platform that contains real and high quality content, high user active ratio, zero distance between developer and users.

Motion是第一个以视频内容和用户认可为主的全球区块链游戏分销平台。Motion平台致力于为开发者提供一个游戏分发平台，该平台包含真实和高质量的内容，高用户活动率，开发者和用户之间零距离。

Motion provides standardized digital currency, multiple open api and sdk for game’ s integration. It is open to welcome all the developers to integrate. Motion delivers the games recommendation authority to the users and users can acquire Motion’ s standardized MTN digital currency through their own active behavior. We hope to let those who contributed to the platform can feel the value and enjoy the great return of Motion.

Motion为游戏集成提供标准化的数字货币、开放的api、sdk。欢迎所有开发人员使用。Motion向用户提供游戏推荐授权，用户可以通过自己的主动行为获得Motion的标准化MTN数字货币。我们希望让那些对平台有所贡献的人能够感受到它的价值并享受Motion的巨大回报。

以上是官网上的简单介绍。

第4章 Motion (MTN) 到底是什么

Motion 是一个结合区块链技术的精品游戏推荐平台，通过短视频和图文内容服务全球高质量玩家，解决游戏资产中心化，游戏商业化系统不透明和平台推荐商业气息严重等行业问题。平台将用户行为价值化，激励玩家反馈游戏建议，帮助游戏开发者更好的优化产品。以用户需求为核心推荐各类玩家喜欢的精品游戏，游戏账户去中心化，保证用户的资产权益。

Motion 平台为全球开发者提供互动社区，账号容器，道具交易，对战系统等平台化服务。底层核心主链秉承用户体验优先的产品理念，连接用户和开发者。开发者可通过 Motion sdk 快速与主链服务完成对接，减少游戏厂商开发成本，帮助游戏更好的聚焦于游戏自身。从用户的需求出发，与全球广大从业者一起维护建设良性的游戏生态。

第5章 业务模式

1. Motion Blockchain-based Games Platform

We are dedicated to develop a globalized blockchain-based games platform. Users can find and actively spread the content through games video distribution. The platform incents the active users with value return. It’ s open to serve for the global developer and thus to promote blockchain technology’s fast growing and development.

基于Motion Blockchain的游戏平台

我们致力于开发全球化的区块链游戏平台。用户可以通过游戏视频分发查找并主动传播内容。平台通过价值回报吸引活跃用户。它可以为全球开发商服务，从而促进区块链技术的快速发展。

2. Game Item Exchange

Motion provides an item exchange platform for all the games. The transparent users assets, game item trading and circulation help to build up a more real game virtual world.

游戏物品交换

Motion为所有游戏提供物品交换平台。透明的用户资产，游戏物品交易和流通有助于建立更真实的游戏虚拟世界。

3. Decentralized Battle Platform

Smart Contract based, decentralized system of witness and arbitration, equitable and fair game battle competition platform.

去中心化的战斗平台
基于智能合约，去中心化的见证和仲裁系统，建立一个公平公正的游戏战斗竞赛平台。

4. Digital Assets Wallet

We create the functional digital assets wallet for global gamers to realize the unified management of digital assets of all the games. Users can truly “own” the game assets through binding.

数字资产钱包
我们为全球玩家创建数字资产钱包，实现所有游戏数字资产的统一管理。用户可以通过绑定从而真正“拥有”游戏资产（比如不能被封号？）。

第6章 MTN 有什么用途？

MTN 是 Motion 平台上的数字资产，开发者和游戏用户都可用 MTN 创造价值：

对于开发者：

  1. 在平台开发、认证应用、使用链上服务需要支付的燃料费用代币。
  2. 开放平台搭建完成后，开放广告投放，平台增值服务，游戏联运等业务板块。会产生部分平台佣金。Motion 团队会定期拿出部分佣金收入，对 MTN 进行不定期回购。
  3. 可购买 MTN，用于激励用户在社区的良性行为

对于用户：

1）作为社区内容的创造者，以及社区的互动者可获得对应的 MTN 激励；
2）用 MTN 使用平台增值服务。
3）用 MTN 对平台流通性道具进行交易。
4）用 MTN 参与平台对战竞赛或其他社区活动。

从上面的描述可以看出，MTN的用途目前来看不是很清楚，短期内增值不大。

第7章 时间线

官网给出的时间线非常不明确，除了2018年2月给出“基础设施正处于原型开发阶段”，其余时间节点未给出相关描述。

第8章 合作伙伴

目前 Motion 项目已经与世纪华通，昆仑万维，恺英网络，边锋科技，天神娱乐等多家公司达成了初步的合作意向，并得到了几十家游戏公司 ceo 的共同支持。

big.one上的描述说只是达成了初步的合作意向。由于官网上已经将这个几家工作的logo放上去，所以假设已经达成合作。

我们来看看这几家公司：

昆仑万维，独角兽挖掘机。趣分期，随手记，洋钱罐都是其投资项目。而游戏方面，拿下多个超热IP进行游戏改编，比如《BLEACH》、《终结者2：审判日》。

实际控制人，周亚辉。在群雄割据的网络游戏中，他用网页游戏打差异化，投入大量资金打造《三国风云》，至此昆仑万维立足并开始飞速发展，而青涩的小伙已经成长为投资圈的大鳄。

恺英网络，网页游戏包场大户。通过旗下的网页游戏平台打通移动互联网的各个流量接口，并以游戏为基础，向游戏分发和影视方向发力，未来规划的VR/AR也是大风口。

世纪华通，跨界并购盛大。从汽车零部件生产商到全球游戏注册用户20亿。曾经的中国首富陈天桥带领的盛大也难逃其并入游戏版图的决心。旗下包括天游、七酷、点点互动、盛大游戏，推动互联网游戏行业的整合和收购。

天神娱乐，花234W美刀跟巴菲特吃个饭。通过收购和投资，天神娱乐已经形成游戏研发与发行、IOS应用分发平台、移动互联网广告平台、影视制作与营销等全方位的泛娱乐产业布局。给人感觉就是特别大，特别厉害。

以上材料来自 “币须说”。

看起来还不错，四大上市游戏公司。

第9章 结束语

使用Site Spider跑了一下官网，发现官网真心简单，除了主页啥也没有了。

也没有找到其他官方信息渠道。

个人认为MTN的愿景是不错的，本人是比较喜欢玩游戏的，Motion可以让游戏开发人员、测试人员、玩家、分发者等都获益。

但是开发难度较大，考虑到该项目的开发团队已经更换（说相声的冯晓东同志目前去开发Mixin了），而且上线big.one之后，基本没有任何宣传，目前该项目属于“画饼”阶段。
开发团队有哪些人、技术能力是否可行、新版白皮书何时能发布、代码库是否公开等存在很大的不确定性。

MTN从上线1小时后到目前为止一直处于破发状态！

因此，慎重参与。

所有相关介绍均来自于第三方并有可能存在错误和遗漏。
区块链资产投资是高风险行为，您必须自行承担价格有可能归零的风险，请根据自己的能力谨慎投资。
