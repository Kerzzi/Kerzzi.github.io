错误记录 | 如何修改Model的栏目名称？'

By Kerzzi

发表于 2017-04-14

文章目录

微信群里讨论的一个问题，比较实用，参考陈俊鸿同学的博客。

动作
创建表的时候，手残把栏目名敲错了rails g model post titlr:string content:text。

分析
把栏目名从titlr改成title

解决方法
step1:
执行rails g migration FixColumnName创建migration

step2:
修改db/xxxxxx_fix_column_name.rb

step3:
执行rails db:migrate，done！
